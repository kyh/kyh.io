# Kwadrants

> 2x2 matrix canvas editor.

Interactive canvas app for creating quadrant diagrams with draggable tags, images, editable axis labels, and PNG/JPEG export.

## Stack

- Framework - [Vite](https://vitejs.dev/) + [React](https://react.dev/)
- Canvas - [Konva](https://konvajs.org/) + [react-konva](https://github.com/konvajs/react-konva)
- Animation - [Motion](https://motion.dev/)
- Styling - [Tailwind CSS v4](https://tailwindcss.com)

## Development

```bash
pnpm install
pnpm dev-kwadrants
```
